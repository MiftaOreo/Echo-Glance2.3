🔧 Mode Perangkat

💤 MPU6050 (Kepala) — Non-aktif, hemat daya
💤 FSR402 (Pipi) — Non-aktif, hindari alarm palsu
● Mode Aktif (Sesi terapi / kegiatan)
✅ TCRT5000 (Mata) — Aktif penuh
✅ MPU6050 (Kepala) — Deteksi >15° → notifikasi
✅ FSR402 (Pipi) — Ukur amplitudo & frekuensi terapi

🤖 Klasifikasi Pola Kedipan

Versi awal menggunakan pendekatan rule-based (state machine) pada ESP32 — sudah cukup akurat untuk pola yang jelas bedanya. Data logging dikumpulkan untuk melatih model Random Forest / Decision Tree dari fitur turunan (durasi rata-rata, interval, jumlah kedip per sesi) guna personalisasi per pasien. Model dilatih di server (Python + scikit-learn) → parameter diekspor ke ESP32.
